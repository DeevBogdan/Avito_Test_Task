# Задание 1: Скриншот с багами
  Цели:
  - Найти баги.
  - Приоритезировать .
  - Перечислить списком.
# Приоритеты:
  Р0 - критический,
  Р1 - высокий,
  Р2 - средний,
  Р3 - низкий.

# Предисловие
    об. - объявление.
    Р0 багов не обнаружено - невозможно затестить, используя скриншот.

# Баги:
  1. Ошибка функционала кнопки "Избранное" (всплывающее окно "Не показывайте мне похожие объявления") - Р1.
  2. Вылетает ошибка в правом верхнем углу - Р1.
  3. Найдено 19 объявлений, выдает 8 - Р2.
  4. Объявление из Новосибирска, вместо Москвы (1-ое об.) - Р2.
  5. Пол не совпадает с фильтром "Мальчик" (2-ое об.) - Р2.
  6. Порода не совпадает с фильтром "Персидская" (6-ое об.)- Р2.
  7. Цена не совпадает с фильтром "1000-30000" (2-ое об., 6-ое об.) - Р2.
  8. Плюшевый котёнок - Р2.
  9. "Сначала из Можайска" вместо "Сначала из Москвы" - Р3.
  10. "Щенок" вместо "Котёнок" - Р3 (Имя не то, но функционал не страдает — выдает котят).
  11. Нарушение сетки (сдвиг об. в правую сторону) - Р3.
  12. Бенгал в объявлении «персидский котёнок» (ошибка модерации) - Р3.
  13. Опечатка: "Корьера" вместо "Карьера" - Р3.
  14. Подборка из бенгалов - Р3 (или ошибка в данных).
  15. Неправильная последовательность подкатегорий:"Главная-Животные-Персидская-Кошки" вместо "Главная-Животные-Кошки-Персидская" - Р3.
  16. Опечатка: "Уведомять" вместо "Уведомлять" - Р3.

# Особенности, которые нельзя назвать багом без требований:
  1. Отсутствуют фильтры по окрасу - в проде есть.
  2. Не у всех объявлений есть время - в проде тоже самое.
